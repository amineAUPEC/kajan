## serveurs / VM :

### www : 
- à installer :  
`sudo apt-get update -y`  
`sudo apt-get install -y apache`  
`sudo apt-get install –y php7  ` 
- avec fichier :
    - ping.php 
    - logs.php
### poste :
- à installer :  
`sudo apt-get update -y`    
`sudo apt-get install -y iptables`  
`sudo apt-get install -y openssh-server`



#### pirate :

`sudo apt-get install -y tcpdump`
